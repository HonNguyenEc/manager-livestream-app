# New Chat Mandatory Context

Use this block as the **first message** whenever starting a new chat with Cline or any API-based coding agent.

## Mandatory instruction

Before doing any task, you must read and apply all governance files in this repository:

- `.cline/rules/*`
- `prompts/*`
- `docs/*`
- `cline_ruleset_full.md`

Then provide a short confirmation containing:

1. Which rule sources were loaded.
2. The consolidated execution policy you will follow.
3. A statement that implementation will start only after this confirmation.

Do not implement anything until this confirmation is completed.
